# Getting Started: OKX Wallet

## What is OKX Wallet?

OKX Wallet is a versatile, non-custodial cryptocurrency wallet designed for managing assets across multiple blockchains, including Ethereum, Bitcoin, Solana, and other major networks.&#x20;

It supports a wide range of use cases, such as trading, staking, NFT management, and interacting with decentralized applications (dApps).&#x20;

* The wallet is available as a mobile app, browser extension, and web wallet.

{% hint style="info" %}
You can download OKX wallet from their official website here:

[https://www.okx.com/download](https://www.okx.com/download)
{% endhint %}
